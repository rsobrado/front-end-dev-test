JavaScript/NodeJS

For this test, you'll write two functions exposed as methods of a node module.

The first method (isPalindrome) should validate whether or not a given string is a palindrome. (assume ascii text and numbers, but the code should ignore non alphanumeric characters, as in evaluating sentences which may contain punctuation spaces and capitalization).

The second method (randomAsyncStringIsPalindrome) will return whether or not a random unknown string (from an asynchronous service) is a palindrome.

isPalindrome

This method should take a string as an argument. That method should:

  • Return true when that String is a palindrome
  • Return false when that String is not a palindrome
  • Return false when that String is empty
  • When passed a parameter that is not a String, throw an Error, or an object that is an instance of one, with a descriptive message about what went wrong

randomAsyncStringIsPalindrome

This method should take an Object as an argument (you do not have to implement this Object - its documentation is below: MysteryStringObjectGetter), and using the interface it exposes fetch a string and return a Promise that will resolve either true or false, indicating whether or not the fetched result is a palindrome.

Documentation for MysteryStringObjectGetter

  • get()
    makes an asynchronous http request and returns a Promise that will resolve to a String if the request is successful.
